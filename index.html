Start-up

Update packages

sudo pacman -Syu

Remove packages

sudo pacman -R [package name]

Remove package caches

sudo pacman -Scc

Restart system

sudo restart

Compile & Execute

Install GIT

sudo pacman -S git

GIT version

git --version

git version

Setup config

git config --global user.email "[user email]"

git --global user.name "[name for commits ,not username]"

Clone repo

git clone [github https link]

Check new changes

git status

Save changed file

git add [filename]

Commit changes locally

git commit -m "[commit comment]" [filename]

Push changes on remote

git push -u origin [branch name]
