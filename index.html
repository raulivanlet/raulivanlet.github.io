Start-up

Update packages

sudo pacman -Syu

Remove packages

sudo pacman -R [package name]

Remove package caches

sudo pacman -Scc

Restart system

sudo restart

Install other apps

sudo pacman -S virt-manager

sudo pacman -S gwenview

sudo pacman -S okular

sudo pacman -S libreoffice-still

sudo pacman -S virtualbox

sudo pacman -S vlc

GIT/Github

Install GIT

sudo pacman -S git

GIT version

git --version

git version

Setup config

git config --global user.email "[user email]"

git config --global user.name "[name for commits ,not github username]"

Clone repo

git clone [github/AUR https link]

Check new changes

git status

Save changed file

git add [filename]

Commit changes locally

git commit -m "[commit comment]" [filename]

Push changes on remote

git push -u origin [branch name]

Build from Source

Install

sudo pacman -S --needed base-devel git

Clone App

git clone [url]]

Navigate to Folder and Print Files

cd [folder]/[folder]/[folder] && ls

Build from Source

less PKGBUILD

makepkg -si

Compile and Execute

Install C/C++ Language Packages

sudo pacman -Syu gcc

Navigate to Folder and Print Files

cd [folder]/[folder]/[folder] && ls

Compile C++ Files

g++ -o [executable file] [fileToBeCompiled.cpp]

Compile C Files

gcc -o [executable file] [fileToBeCompiled.c]

Execute the Executable File

./[numeFisierExecutabil]
