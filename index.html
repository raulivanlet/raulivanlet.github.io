Install Arch

Show Disks and Partitions

lsblk

Enter Partition Tool in Disk sda or nvme0n1

cgdisk /dev/sda or nvme0n1

  • 500M ef00 boot
  • 20G 8200 swap
  • ALL 8300 arch

Format Partitions

mkfs.fat -F32 /dev/sda1
mkswap /dev/sda2
mkfs.ext4 /dev/sda3

Mount Partitions

mount /dev/sda3 /mnt
swapon /dev/sda2
mkdir -p /mnt/boot
mount /dev/sda1 /mnt/boot

Preinstall Packages

pacstrap /mnt base base-devel linux linux-firmware vim networkmanager

Create drives linkin points

genfstab -U /mnt >> /mnt/etc/fstab

Boot into arch instalation

arch-chroot /mnt

Set local time

ln -sf /usr/share/zoneinfo/Europe/Bucharest /etc/localtime

Set local date format , currency ,etc

vim /etc/locale.gen

en_US.UTF-8 UTF-8

Set language

vim /etc/locale.conf

LANG=en_US.UTF-8

locale-gen

Create root user

vim /etc/hostname

root

passwd

Initial RAMdisk

mkinitcpio -P

Install GRUB

pacman -Sy grub efibootmgr
grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB
grub-mkconfig -o /boot/grub/grub.cfg

Enable Wifi / Ethernet

systemctl enable NetworkManager

Add non-root User

useradd -g users -G wheel,storage,power -m [username]
passwd [username]

Enable Non-Root User To Use SUDO

visudo

##Uncomment to allow members of group wheel to execute any command
%wheel ALL=(ALL:ALL) ALL

Finished Basic ARCH Install

exit
umount -R /mnt
reboot

GIT/Github

Install GIT

sudo pacman -S git
git version

Setup config

git config --global user.email "[user email]"
git config --global user.name "[name for commits ,not github username]"
git config --global credential.helper cache
git config --global credential.helper 'cache --timeout=7200'

Clone repo

git clone [github/AUR https/SSH link]

Check new changes

git status

Save all changes

git add -A

Add changes to local repo

git commit -m "Updated for ..."

Add changes to cloud repo

git push
