Install Arch

Show Disks and Partitions

lsblk

Enter Partition Tool in Disk sda or nvme0n1

cgdisk /dev/sda or nvme0n1

  • 500M ef00 boot
  • 4G 8200 swap
  • ALL 8300 arch

Format Partitions

mkfs.fat -F32 /dev/sda1
mkswap /dev/sda2
mkfs.ext4 /dev/sda3

Mount Partitions

mount /dev/sda3 /mnt
swapon /dev/sda2
mkdir -p /mnt/boot
mount /dev/sda1 /mnt/boot

Preinstall Packages

pacstrap /mnt base base-devel linux linux-firmware vim networkmanager

Create drives linkin points

genfstab -U /mnt >> /mnt/etc/fstab

Boot into arch instalation

arch-chroot /mnt

Set local time

ln -sf /usr/share/zoneinfo/Europe/Bucharest /etc/localtime

Set local date format , currency ,etc

vim /etc/locale.gen

en_US.UTF-8 UTF-8

Set language

vim /etc/locale.conf

LANG=en_US.UTF-8

locale-gen

Create root user

vim /etc/hostname

hostname

passwd

Initial RAMdisk

mkinitcpio -P

Install GRUB

pacman -Sy grub efibootmgr
grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB
grub-mkconfig -o /boot/grub/grub.cfg

Enable Wifi / Ethernet

systemctl enable NetworkManager

Add non-root User

useradd -g users -G wheel,storage,power -m [username]
passwd [username]

Enable Non-Root User To Use SUDO

visudo

##Uncomment to allow members of group wheel to execute any command
%wheel ALL=(ALL:ALL) ALL

Arch CLI install finished , can skip DE

Install Desktop Environment

pacman -Syu

Install OpenGL:

[AMD] pacman -S mesa [lib32-mesa]
[NVIDIA] pacman -S mesa nvidia-utils [lib32-nvidia-utils]
[INTEL] pacman -S mesa

Install Vulkan:

[AMD] pacman -S amdvlk lib32-amdvlk
[NVIDIA] pacman -S nvidia-utils lib32-nvidia-utils
[INTEL] pacman -S vulkan-intel lib32-vulkan-intel

AMD Drivers:

pacman -S xf86-video-amdgpu

Intel Drivers:

pacman -S xf86-video-intel

Nvidia Drivers:

pacman -S nvidia

pacman -S xorg-server xorg-xrandr i3-wm i3bar i3status konsole dolphin chromium

pacman -S xorg-server plasma-desktop konsole dolphin chromium
systemctl enable sddm

Finished Basic ARCH Install

exit
umount -R /mnt
